This Website

This is my portfolio website. The purpose of this website was to showcase the work I've done since I started heavy into web development. I used a mobile first design approach as I consider this to be more practical in the present context by way of the Progressive Advancement concept. It was my biggest challenge yet.

For this website I gained even more familiary using SCSS, which I now consider to be an indispensable tool. As for JavaScript, I decided that the best way to go for now was vanilla, so that I can develop a deeper understanding before taking on a framework like React. Every component is custom made, which includes:

  • Navbar functionality & background efect

  • Project gallery filter

  • Multiple modals with possibility of adding more without adjusting JS code

  • Carousel slider

  • Page sections animations on scroll

Natours

Natours is a mock landing page for a travel agency, specializing in nature tours. The purpose of this project was to exercise in creating a single-page website using advanced HTML and CSS for its development with the following bullet points:

  • CSS techniques to create advanced designs and effects

  • Advanced CSS animations with @keyframes, animation and transition

  • CSS architecture: component-based design, BEM, writing reusable code

  • Float layouts. Although not a modern approach, the purpose was to better understand the way older sites have been made

  • Using Sass: global variables, architecting CSS, managing media queries

  • Advanced responsive design: media queries, mobile-first vs desktop-first

  • Responsive images in HTML and CSS for faster pageloads

  • SVG images and videos in HTML and CSS: building a background video effect

  • The NPM ecosystem: development workflows and building processes

This was part of the Udemy Advanced CSS and Sass Course by Jonas Schmedtmann.

  • Date: December, 2021

Dice Game

This project is a two player share screen game in which you have to roll the dice in order to accumulate points. Whoever reaches 100 points first wins. If you roll the number one, your current points get reseted and you lose your turn.

This project helped me further practice the JavaScript, DOM and Events fundamentals.

Martial Arts website

This is a real project and is used by my martial arts school. I finished this website in September 2020 as a means to promote and advertise the school. It is now ranked as the first school of this type on google search results for "Wing Chun Kung Fu Bucuresti" and second for "Wing Chun Bucuresti".

The website was built using WordPress in a time where I was not that interested in pursuing a career in Web Development and subsequently having no knowledge in this regard. However I managed to build it using the Avada Website Builder for WordPress and successfully implemented a membership plugin that enables additional written and video learning resources to be seen by users who have multiple tiered accounts.

Besides learning and using WordPress, I also gained familiarity for implementing APIs with Google Maps, Stripe and Paypal.

The website is fully responsive, however it needs further optimization for mobile performance.

My goal for the future is recreating the website without WordPress and changing the design whilst keeping similar functionalities or adding more (e.g. online courses).
